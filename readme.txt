Accept payments using your Paywall merchant account with this WooCommerce plugin.

== Description ==
This plugin integrates Paywall Checkout with WooCommerce to allow merchants to accept payments using their Paywall merchant account. With this plugin, you can enhance your WooCommerce store's checkout process and provide a seamless payment experience for your customers.

== Installation ==

Download the plugin zip file.
Upload the plugin to your WordPress site's wp-content/plugins/ directory.
Extract the zip file in the wp-content/plugins/ directory.
Activate the plugin through the WordPress admin panel.
Go to the WooCommerce > Settings > Payments page and enter your Paywall account details.
